Frequently Asked Questions?

Learn about the CitiDient Web App from our curated frequently most asked questions.

Q: What is the INEC IReV portal?

A: The INEC IReV portal is the online platform used by the Independent National Electoral Commission (INEC) of Nigeria to publish and share election results.

Q: What is the purpose of the CitiDient Web App?

A: The CitiDient Web App is an open source web app written in python flask that aims to help candidates in the just concluded general elections use their communities to help transcribe election results online. The app is designed to facilitate the transcription of result sheets by registered users, allowing them to enter votes, accreditation data and verify signatures and stamps.

Q: What is the MIT Licence?

A: The MIT Licence is a permissive free software licence that permits users to use, modify and distribute the software with few restrictions. The licence is often used for open source software.

Q: Is CitiDient Web App available under the MIT Licence?

A: Yes, the CitiDient Web App is available under the MIT Licence. This means that users can use, modify and distribute the software with few restrictions.

Q: How do I use CitiDient Web App?

A: To use the CitiDient Web App, you must first register as a user on the app. Once you have registered, you will be presented with result sheets that you can transcribe by entering votes, accreditation data and verifying signatures and stamps.

Q: Can I access the CitiDient Web App from any device?

A: Yes, the CitiDient Web App is a web app and can be accessed from any device with an internet connection and a web browser.

Q: Is the CitiDient Web App affiliated with the Independent National Electoral Commission (INEC) of Nigeria?

A: No, the CitiDient Web App is not affiliated with the Independent National Electoral Commission (INEC) of Nigeria. The app is designed to facilitate the transcription of result sheets by registered users, but it is not officially endorsed or sponsored by INEC.

Q: Can I modify and redistribute the CitiDient Web App?

A: Yes, the CitiDient Web App is available under the MIT Licence, which permits users to modify and redistribute the software with few restrictions.

Q: Is there a cost to use the CitiDient Web App?

A: No, the CitiDient Web App is a free and open source software that is available for anyone to use at no cost.
